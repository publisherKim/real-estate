## 취득세
- 취득
    - 사실상 
        - 원시
            - 토지: 공유수면 매립간척
            - 건축물: 신축, 증축, 재축
            - 선박: 건조
            - 차량, 기계장비, 항공기: 제조, 조립
            - 광업권, 어업권: 출원
            - 민법상 시효취득
        - 승계취득
            - 유상승계취득: 매매, 교환, 현물출자 등
            - 무상승계취득: 상속, 증여, 기부
    - 의제취득
        1. 토지: 지목변경(가액의 증가)
        2. 건축물: 개수(대수선 + 시설물의 설치)
        3. 차량, 기계장비, 선박: 종류변경(가액의 증가)
        4. 과점주주(50%의 초과)의 주식취득(단, 법인 설립시는 제외)
    - 단, 차량, 기계장비, 선박, 광업권, 어업권은 승계취득시 과세
### 취득시기 ··· 언제취득: 60일내 신고·납부
1. 유상(매매) ··· 잔금지급일
    - 객관적 취득
        - 입증: 국가등·수입·공(경)매·판결·법인장부
    - 일반적 취득
        - 원칙: 계약상 잔금 지급일
        - 예외: 계약일로부터 60일이 경과되는 날
2. 연부취득 ··· 매년지급
    - 연부기간 2년 이상, 취득가액 50만원 초과
    - 원칙: 사실상 연부금 지급일
    - 예외: 이전에 등기 ··· 등기·등록일
3. 무상
    1. 증여
        - 원칙: 증여 계약일
        - 예외: 이전 등기 ··· 등기·등록일
    2. 재산분할: 등기·등록일
    3. 상속·유증: 상속개시일
4. 종류변경 ··· 사실상·공부상 변경된 날 중 빠른 날
5. 신축
    1. 일반(건축): 사용승인서·임시사용 승인일·사실상 사용일 중 빠른 날
    2. 도·개·법: 준공 검사 증명서, 도·정·법: 준공 인가증 또는 임시사용 승인일 과 사실상의 사용일 중 빠른 날
### 과세표준 * 세율 = 산출세액
1. 원칙: 취득당시 신고가액 = 취득가액
2. 예외: 시가표준액 ··· 정부
    - 무신고 또는 신고가 없을 때
    - 신고가액이 정부가 정한 시가표준액보다 적을 때
3. 특례: 사실상 취득가액 ··· 실제 지급액
    - 입증: 국가등·수입·공(경)매·판결·법인장부·검증
        - 단, 무상취득이나 부당행위는 제외
4. 의제취득: 변경 + 가액 ··· 증가액
    - 원칙: 신고가액
    - 예외: 시가표준액
5. 법인이 아닌 자가 건축물을 건축한 경우
    - 취득가격 중 100분의 90을 넘는 가격이 법인장부에 따라 입증된 금액
        1. 법인장부로 입증된 금액
        2. 계산서 또는 세금 계산서로 입증된 금액
        3. 국민주택 채권을 취득 이전에 양도 함으로써 발생하는 매각 차손
- 취득 가액 포함되지 않는 금액
    - 개인: 할부, 중개수수료, 연체로
    - 법인: 광고선전비, 분담비, 이주비, 보상금, 부가가치세
### 세율
- % 정율세율
    - 차등비례세율·일부 단순누진세율
- 표준세율
    - 기본
    - +- 50% 가감
- 특례세율
- 중과세율
### 표준세율
- 유상
    - 농지: 3%
    - 기타: 4%
    - 주택
        - 6억원 이하: 1%
        - 6억 초과 ~ 9억원 이하: 공식
        - 9억초과: 3%
- 상속
    - 농지 2.3%
    - 기타: 2.8%
- 증여
    - 비영리자: 2.8%
    - 일반: 3.5%
- 원시(신축·증축·매립간척): 2.8%
- 분할: 2.3%
### 특례
#### 표준세율 - 2%
- 환매 ··· 주택: 세율 * 50%
- 상속
    - 감면농지 = 자경농지 2.3 -2 = 0.3%
    - 1가구 1주택 ··· 2.8 - 2 = 0.8%
- 법인의 합병 ··· 3.5 - 2 = 1.5%
- 분할 ··· 2.3 - 2 = 0.3%
- 건축물의 이전 ··· 2.8% - 2 = 0.8%
    - 단, 종전건축물 가액을 초과하는 부분은 제외한다.
- 이혼시 분할 ··· 3.5 - 2 = 1.5%
- 입목의 취득
#### 2% ··· 중과기준세율
- 개수(면적 증가는 제외)
- 변경
- 과점주주
- 수입
- 레저
- 묘지
- 임시건축물 1년 초과시
#### 중과세율
- 사치성재산 ··· 별장, 골프장, 오락장, 고급선박, 고급주택
    - 표준세율 + 8%
        - 중과기준세율(2%) * 400/100
    1. 별장 ··· 건축물 + 부속 토지(토지 경계 불분명 시 건축물의 바닥면적에 10배)
        - 주택 -> 상시주거 X -> 휴양(오피스텔도 포함)
    2. 골프장 ··· 토지 + 건축물 + 입목
        - 회원제
        - 원시취득
            - 등록(변경등록)
            - 사용
    3. 고급오락장 ··· 건축물 + 부속토지(토지 경계 불분명 시 건축물의 바닥면적에 10배)
        - 용도변경 시 제외
    4. 고급선박
    5. 고급주택 ··· 건축물 + 부속토지(토지 경계 불분명 시 건축물의 바닥면적에 10배)
        - 단독주택
            1. 건물면적 331㎡ 초과 + 개별주택가격 9억 초과
            2. 대지면적 662㎡ 초과 + 개별주택가격 9억 초과
            3. 엘리벵이터 + 개별주택가격 9억 초과
            4. 에스컬레이터 또는 수영장
        - 공동주택
            - 일반: 245㎡초과 + 공동주택가격 9억 초과
            - 복층: 274㎡초과 + 공동주택가격 9억초과
        - 용도변경 시 제외
- 과밀억제권역 안 법인의 부동산 취득
    - 표준세율 + 4%
    - 과밀 + 법인의 부동산 취득
        - 모든 ··· 영리·비영리, 본점·주사무소 + 원시취득 + 사업용
    - 과밀 + 공장의 신·증설
        - 서울·인천·경기도 등
        - 신설 ··· 생산설비 + 500㎡ 이상
        - 증설 ··· 20/100 이상 + 330㎡ 초과
        - 비도시형 업종
        - 대상 ··· 건축물 + 부속토지 + 차 + 기계장비
- 대도시 안 법인 설립등에 따른 부동산 취득
    - (표준세율 * 3) - 4%
    - 대도시 + 법인의 부동산 취득
        - 모든 ···영리·비영리
        - 설립·전입 전에 취득: 본점(주사무소)·지점(분사무소) + 원시·승계 + 사업용
        - 설립·전입 후에 취득: 본점(주사무소)·지점(분사무소) + 원시·승계 + 사업용·비사업용
    - 대도시 + 공장 신·증설
        - 상동
        - 대상 ··· 건축물 + 부속토지
- 비도시형업종은 중과대상 <-> 도시형업종은 중과대상 제외
- 주택 중과 ··· 4%
    - 표준세율(주택 1채)
        1. 6억 이하 ··· 1%
        2. 6억 초과 ··· 공식
        3. 9억초과 ··· 3%
    - 일반지역
        - 1채 + 1채: 1% * 공식 * 3%
        - 2채 + 1채: 4% + 4%(2% * 200/100)
        - 3채 + 1채: 4% + 8%(2% * 400/100)
    - 조정지역
        - 1채 + 1채: 4% + 2%(중과기준세율)
        - 2채 + 1채: 4% + 4%(2% * 200/100)
        - 3채 + 1채: 4% + 8%(2% * 400/100)

### 납부
- 원칙 ··· 신고·납부
- 일반적: 60일내 신고·납부
- 상속
    - 달의 말일부터 6개월 내 신고·납부
    - 외국에 거주시 9개월 내 신고·납부
- 허가구역: 허가, 해제일, 축소일 후 60일 내 신고·납부
- 등기·등록시: 등기·등록관서 접수하는 날까지
#### 특례
- 일반 -> 중과세
    - 일반재산 취득 후 5년 내 중과대상으로 변경된 경우: 60일 내 신고·납부
- 비과세 감면 -> 추징: 추징사유 발생일로부터 60일 내 신고·납부
### 부가세
- 납부
    - 농어촌특별세 ··· 취득가 * 2% = 산출세액 -> 10%
    - 지방교육세
        - 주택 유상 ··· 취득가 * (세율 * 50%) = 산출세액 -> 20%
        - 이외 ··· 취득가 * (표준세율 - 2%) = 산출세액 -> 20%
- 감면: 20% 농어촌특별세
