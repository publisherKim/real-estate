    - 부령
        - ~법 시행규칙
#### 조례
    - 지자체

        - 기초

# 중앙
    - ~부, ~장관 = 국가
## 지방
    - 광역
        - 특별시(구), 광역시(구, 군), 특자시, 도(시, 군), 특자도
        - 시, 도
    - 기초
        - 시, 군, 구
    - 인구 50만
        - 미만: 시
        - 이상: 대도시

### 행정 공법 체계
    - 국장 => 국가
    - 시, 도지사 => 광역
    - 시, 군, 구 => 기초
    - 시, 도 / 대도시 시장(광역 + 대도시)
    - 특자시, 특자도, 시, 군, 구 => 기초 업무 수행
    - 특시, 광시, 특자시, 특자도, 시, 군 => 하나의 도시

# 개발 
## 과거
### 도시 
    - 도시계획법
    - 도시계획에 따른
    - 개발 시행
    - 난개발 발생 없음
### 비도시
    - 국토이용관리법
    - 도시계획 없음
    - 개발 시행
    - 난개발 발생함
## 현재
    - 난개발을 방지하기 위해 국계법 출현

### 국토의 계획 및 이용에 관한 법률
    - 총설: 목적, 용어
    - 계획 
        - 광역도시계획
        - 도시(군)기본계획
        - 도시(군)관리계획
            - 군은 시골에 해당
    - 이용(은 개발은 규제대상)
        - 개발행위
            - 건축물의 건축
            - 공작물 설치
            - 토지의 형질변경
            - 토석의 채취
            - 토지분할
            - 물건을 쌓아놓는 행위
        - 허가절차
        - 기반시설 설치
