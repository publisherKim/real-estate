## 투자분석 기법
### 현금흐름의 측정
- 영업수지(소득이득)
    - 단위당 예상 임대료 * 임대단위 수
    - 가능총(조) 소득 - 공실 및 불량부채 + 기타소득
    - 유효총(조)소득 - 영업경비
        - 포함 (o)
            - 유지관리비, 수도전기료, 화재보험료, 광고료, 재산세, 수수료 등
        - 포함(x)
            - (-) 항목: 공실, 부채서비스액, 영업소득세
            - 감가상각비
            - 개인업무비, 소유자 급여
    - 순영업소득 - 부채서비스액
    - 세전 현금수지(흐름) - 영업소득세
    - 세후 현금수지(흐름)
- 지분복귀액(자본이득)
    - 예상 매도가격 - 매도경비
    - 순매도액 - 미상환 저당잔금
    - 세전지분복귀액 - 자본이득세
    - 세후지분복귀액
- 영업소득세
    - 순영업소득 + 대체충당금 - 이자지급액 - 감가상각액
        - 절세효과
            - 투자과정에서 대출금에 대한 원리금 지불액 중 이자비용은 비용인정
            - 건물에서 수익이 발생 동안 나나탈 수 있는 감가상각은 비용 인정
    - 과세대상소득 * 세율
    - 영업소득세
### 할인현금흐름(수지)분석법
- 순현가법(NPV: Net Present Value)
    - 순현가: 현금유입의 현가 - 현금유출(지분)
        - 현금유입의 현가 - 현금 유출 = +
            - 순현가 > 0 -> 초과이윤 => 투자채택
        - 현금유입의 현가 - 현금유출 = 0 
            - 순현가 = 0 -> 정상이윤 => 투자채택
        - 현금유입의 현가 - 현금 유출 = -
            - 순현가 <> 0 -> 적자 => 투자기각
- 수익성지수법(PI: Profitability Index)
    - 수익성지수 = 현금유입의 현가 / 현금유출(현가)
        - 현금유입의 현가 / 현금유출(현가) > 1 
            - 수익성지수 > 1 -> 초과이윤 => 투자채택
        - 현금유입의 현가 / 현금유출(현가) = 1
            - 수익성지수 = 1 -> 정상이윤 => 투자채택
        - 현금유입의 현가 / 현금유출(현가) < 1
            - 수익성지수 < 1 -> 적자 => 투자기각
- 내부수익률법(IRR: Internal Rate Of Return)
    - 개념: 현금유입의 현가와 현금유출(현가)을 같게 만드는 할인율
        - 15% -> 1억 1,000만원/ (1 + x) = 1억
            - x = 0.1
    - 역활: 순현가를 "0" 수익성지를 "1"로 만드는 할인율
        - 순현가: 1억 - 1억 = 0
        - 수익성지수: 1억/1억 = 1
    - 내부수익률 > 요구수익률 -> 초과이윤 -> 투자채택
    - 내부수익률 = 요구수익률 -> 정상이윤 -> 투자채택
    - 내부수익률 < 요구수익률 -> 적자 -> 투자기각

### 어림셈법 -> 할인(x)
- 승수법
    - 총(조)소득승수 = 총투자액 / 총(조)소득
    - 순소득 승수 = 총투자액 / 순영업소득
        - 자본회수기간은 짧을수록 좋다.
    - 세전현금수지승수 = 지분투자액 / 세전현금수지
    - 세후현금수지승수 = 지분투자액 / 세후현금수지
- 수익률법
    - 종합자본환원율 = 순영업소득 / 총투자액
        - 자본환원율은 수익률이다.
    - 지분배당율 = 세전현금수지 / 지분투자액
    - 세후수익률 = 세후현금수지 / 지분투자액
- 총소득, 순영업소득 -> 총투자액, 세전·세후 현금수지 -> 지분투자액
### 비율분석법
- 대부비율(LTV): 부채잔금(6억) / 부동산가격(10억) = 60%
- 부채비율: 부채잔금(6억) / 지분(4억) = 150%
    - 대부비율이 높아지면 -> 부채비율도 높아진다.
- 부채감당률: 순영업소득 / 부채서비스액
    - 순영업소득으로 부채서비스액을 감당할 수 있는 지표
    - 부채서비스액에 대한 순영업소득의 비율
    - 부채감당률 < 1 -> 소득으로 원리금 상환 불가능(대출 x)
    - 부채감당률 > 1 -> 소득으로 원리금 상환 가능(대출 o)

## 부동산 금융
### 이자율
- 고정 -> 명목이자율 = 실질이자율(위험률 포함) + 예상 인플레율
- 변동 -> 실질이자율 = 실질이자율(위험률 포함)
    - 이자율 크기: 고정금리 > 변동금리
    - 변동이자율 = 기준금리(CD 또는 COFIX) + 가산금리
        - 기준금리 차입자와 무관 -> 동일적용
        - 가산금리 차입자에 따라 차등적용
- 이자율 위험(고정금리)
    - 약정이자율 > 시장이자율 -> 조기상환(위험)
    - 약정이자율 < 시장이자율 -> 인플레 위험
        - 은행은 손해, 차입자는 이익
        - 은행은 변동금리 선호
### 저당의 상환방식
#### 대출: 1억(10%, 고정금리), 기간: 10년(매년 말 1회 상환), 저당상수 : 0.1627
- 원금균등(체감식)상환방식
    - 특징: 매기 균등한 원금 + 감소하는 이자 -> 원리금 체감
- 원리금균등 상환방식
    - 특징: 매기 균등한 원리금 -> 감소하는 이자 + 증가하는 원금
- 체증식상환방식 -> 초기 지불액 낮고, 시간이 지날수록 지불액이 증가
    - 대출초기 부(-)의 상환 발생
### 프로젝트 파이낸싱(PF)
- 개발소요자금 약 3000억, 담보 부동산 물건 가치 100억
    - 개발업자(차입자, 시행사, 사업주)
        - 수익성(사업의 현금 흐름) 담보제공
        - 부외 금융효과
        - 비소구 금융 또는 제한적 소구 금융
    - 은행(대주단, 신디케이트론) 
        - 자금제공
        - 독립적인 법인설립 요구(신용의 절연)
        - 위탁계정(에스크로우)
    - PF에 의해 조달된 자금은 개발업자의 회계상 부채로 반영되지 않는다.
    - 부동산이 담보가 아닌 사업성(수익성, 현금흐름)이 담보
    - 부외 금융효과에 의한 채무수용능력 제고

### 주택연금(역연금저당)
- 개념 및 가입조건
    - 집을 담보로 제공하고 은행으로부터 매월 연금식으로 지급을 받는 행위
    - 자격: 부부 중 1인이 만 55세 이상
- 연금채권의 상환
    - 주택처분가액 > 대출잔액 -> 잔여분 상속인에게 돌려줌
    - 주택처분가액 < 대출잔액 -> 부족분 별도의 청구 없음

### 주택저당채권의 유동화
- 유동화의 개념(효과)
    - 은행의 유동성 증가 -> 대출 확대
    - 은행의 유동성 위험 감소
- 저당시장의 구조
    - 1차 저당 시장
        - 자금대출시장 -> 유동화(x)
    - 2차 저당 시장
        - 자금공급시장 -> 유동화(o)
        - MBS는 안정적이다.(위험이 낮다.)
- 주택저당증권(MBS)
    - 주택저당 집합물
        - 소유(지분)권
        - 원리금수취권
    - MBS
        - (지분형)증권
            - 소유권과 원리금수취권이 투자자에게 있음
            - MPTS(이체증권)
                - 투자자가 모든 권리를 가져감
        - 채권(형)
            - 소유권이 발행자에게 있음
            - MBB(저당담보부채권)
                - 발행기관이 투자자에게 수익률을 보장해주고 모든 권리를 가져감
            - 혼합형: 소유권 발행기관(MBB성격), 원리금 수취권은 투자자(MPTS 성격)
                - MPTB
                - CMO
- 지분형증권과 부채증권
    - 부채금융(갚기로 하고 빌리는 것)
        - MBS
        - ABS
        - 주택상환사채
        - (담보)신탁금융
        - 저당금융
    - 지분금융(지분매각)
        - 신디케이트
        - 조인트벤처
        - 리츠
        - 공모에의한 증자
        - 부동산 뮤츄얼펀등 
    - 메자닌금융: 배당 우선주, 신주인수권부사채, 전환사채 등
### 부동산투자회사(REITs)
- 리츠의 특징
    - 부동산 투자(신탁)회사
    - 주식의 형태
        - 간접·소액투자 가능, 환금성 높다.
        - 부동산투자회사만 합병 가능
    - 리츠의 종류
        - 자기관리 부동산투자회사
            - 실체형
                - 임직원 상근(o), 지점설치(o), 자산의 투자·운용-> 직접
            - 설립시 5억, 영업인가후 70억
            - 영업인가 또는 등록후 최저자본금을 갖춘후 현물출자 가능
        - 위탁관리 부동산투자회사
            - 명목형
                - 임직원 상근(x), 지점설치(x), 자산의 투자·운용-> 자산관리회사에 위탁
            - 설립시 3억, 영업인가후 50억
            - 영업인가 또는 등록후 최저자본금을 갖춘후 현물출자 가능
        - 기업구조조정 부동산투자회사(CR리츠)
            - 명목형
            - 설립시 3억, 영업인가후 50억
            - 영업인가 또는 등록후 최저자본금을 갖춘후 현물출자 가능

## 한 줄 마무리
- 영업경비에는 재산세, 전기료, 감가상각비, 유지관리 등이 포함되지 않는다.
- 순영업소득과 세전혐금수지는 동일할 수 있다.
- 순현가법, 연평균순현가법, 현가회수기간법은 할인율을 적용한다.
- 내부수익률은 순현가를 "0"으로, 수익성지수를 "1"로 만드는 할인율이다.
- 승수 값이클수록 투자한 자본회수기간이 길다.
- 대부비율이 높아질수록 부채비율도 높아진다.
- 부채감당률이 1보다 작으면 순영업소득으로 부채에 대한 할부금을 상환하지 못한다.
- 동일한 조건이라면 고정이자율이 변동이자율보다 높다.
- 대출기간 총 이자의 크기: 체증식 > 원리금균등 > 원금균등
- 프로젝트 파이낸싱에 의한 부채는 사업주(차주, 개발업자)의 회계상 부채로 표시되지 않는다.
- 주택저당채권의 유동화로 은행의 유동성이 증가하고, 유동성위험은 감소한다.
- MBB의 투자자는 조기상환의 위험을 부담하지 않는다.
- 부채금융: MBS, ABS, (담보)신탁금융, 저당금융, 주택상환사채등이 있다.
- 자본금(설립): 자기관리 5억, 위탁관리 3억, 기업구조조정 3억
- 영업 또는 등록 후: 자기관리 70억, 위탁관리 50억, 기업구조조정 50억
